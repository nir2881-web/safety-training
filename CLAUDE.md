# Project Preferences

## Deployment
- This project runs **exclusively in the cloud** (GitHub Pages)
- Do NOT suggest running anything locally (`npm run dev`, `localhost`, etc.)
- All changes must be committed and pushed to GitHub
- The app is deployed via GitHub Actions to GitHub Pages at:
  `https://nir2881-web.github.io/safety-training/`
- Branch for development: `claude/safetyon-lms-app-SChpA`
- GitHub Pages is served from the `gh-pages` branch (auto-deployed by GitHub Actions)
