SafetyOn LMS - תיעוד הפרויקט

תאריך: 17/03/2026

1. מה בנינו

אפליקציית LMS (Learning Management System) לניהול הכשרות בטיחות לעובדים.

שם הפרויקט: SafetyOn LMS

טכנולוגיות:

  • React + Vite - הפרונטאנד
  • Tailwind CSS - עיצוב
  • Firebase Firestore - בסיס נתונים בענן
  • Firebase Hosting - אחסון ופרסום האתר

6. פקודות חשובות

לפריסה מחדש (אחרי כל שינוי בקוד):

npm run build
firebase deploy

להעלות קבצי Firebase ל-GitHub (פעם אחת!):

git add firebase.json .firebaserc
git commit -m "Add Firebase hosting configuration"
git push -u origin claude/safetyon-lms-app-SChpA

להריץ לוקאלית:

npm install
npm run dev

7. Firebase - פרטים
